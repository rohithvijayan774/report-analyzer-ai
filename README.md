# 🧠 Report Analyzer AI API

An AI-powered backend system that analyzes blood test reports and provides health insights including risk levels, BMI, and parameter status.

---

## 🚀 Features

- 📊 Analyze lab values (Hemoglobin, Glucose, Cholesterol, Vitamin D)
- 🧠 Risk scoring system (Low / Moderate / High)
- ⚖️ BMI calculation
- 📄 PDF lab report parsing
- 🌐 REST API using FastAPI
- ☁️ Deployable on Render
